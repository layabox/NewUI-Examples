## NewUI Examples

This repository contains sample scenes and UI components built with the LayaAir engine. It is intended as a reference project to demonstrate how to structure UI pages, windows, widgets, and basic interactions in a LayaAir TypeScript project.

### Installation
Download or clone the repository to your local machine. Ensure you have LayaAir IDE installed to open and run the project.
